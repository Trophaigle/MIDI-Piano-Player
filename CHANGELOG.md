# Changelog

## [1.0.0] - First Release

### Added
- MIDI playback support (requires DryWetMIDI from Unity Asset Store or git)
- PDFs for music sheets
- Animations and AnimatorControllers for interactive objects

### Known Issues
- Tempo may not always sync perfectly with MIDI playback (work in progress)

### Notes
- Make sure to import DryWetMIDI from Unity Asset Store (or git) before using the package
